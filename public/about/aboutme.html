This is Briar!

art by orirising

Heya! I’m Wildcat; not my real name but I’ve been going by it on the Internet for so long it might as well be. I’m a queer acearo probably-woman (she/her) from the USA. I like dragons so you’ll probably see me represent myself as my sona Briar, who’s a dragon lady!

I’m a self-taught artist who’s been drawing since I was very young, though only started drawing seriously when I was an early teenager. I consider myself a casual hobbyist though honestly, art is still a very important part of my life. I’ve also been writing since I was very young, starting with my pet headworld Cosmotalia which I will someday write something coherent set in. With that comes a love for worldbuild, even if I land on the soft side of the spectrum.

My preferred art program is Clip Studio Paint and preferred writing program is LibreOffice Writer. I’m currently using Visual Code Studio for code, like what you see on this website, but am considering switching to its open source counterpart VSCodium.
